# Chatbot Kesehatan Berbasis LLM (Palm2)

Gambar menunjukkan antarmuka **chatbot kesehatan** yang memanfaatkan **Large Language Model (LLM)**, yaitu **Palm2** dari Google. Chatbot ini dirancang untuk membantu pengguna dalam konsultasi kesehatan dengan cara interaktif melalui percakapan berbasis teks.

---

## 1. Panel Kiri: Menu Navigasi
- Panel berwarna biru.
- Menu utama:
  - **Konsultasi** → area untuk memulai interaksi dengan chatbot.
  - **Petunjuk Penggunaan** → panduan agar pengguna memahami cara menggunakan chatbot.
- Terdapat tombol **Clear Chat History** untuk menghapus riwayat percakapan, menjaga privasi dan kebersihan data.

---

## 2. Panel Kanan: Area Chat
- Panel gelap untuk area percakapan.
- Judul halaman: **Chatbot Kesehatan**
- Deskripsi: "Chatbot kesehatan ini menggunakan Palm2"
- Percakapan:
  - Pesan pengguna: *"Ada yang bisa AI bantu?"*
- Terdapat **input box** untuk mengetik pertanyaan atau keluhan, dengan tombol kirim di sebelah kanan.

---

## 3. Teknologi di Balik Chatbot
- Chatbot ini memanfaatkan **Palm2**, salah satu **Large Language Model (LLM)** modern.
- Kemampuan LLM:
  - Memahami konteks percakapan.
  - Memberikan jawaban yang relevan dan informatif.
  - Dapat menangani pertanyaan kesehatan secara umum.
- LLM memungkinkan chatbot untuk meniru interaksi manusia dan memberikan respon adaptif sesuai input pengguna.

---

## 4. Kesimpulan
Chatbot ini menggabungkan antarmuka sederhana dengan kekuatan **Palm2 LLM**, sehingga pengguna dapat melakukan konsultasi kesehatan secara mudah, cepat, dan interaktif melalui percakapan teks.
